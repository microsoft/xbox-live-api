# HCIsInitialized  

Returns true if library has been initialized by HCInitialize  

## Syntax  
  
```cpp
bool HCIsInitialized(  
)  
```  
  
### Parameters  
  
  
### Return value  
Type: bool
  
Returns true if this library has been initialized, otherwise returns false.
  
## Requirements  
  
**Header:** httpClient.h
  
**Library:** libHttpClient.141.GSDK.C.lib
