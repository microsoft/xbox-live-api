# HCRemoveWebSocketRoutedHandler  

Removes a previously added HCWebSocketRoutedHandler.  

## Syntax  
  
```cpp
void HCRemoveWebSocketRoutedHandler(  
         int32_t handlerId  
)  
```  
  
### Parameters  
  
*handlerId* &nbsp;&nbsp;\_In\_  
Type: int32_t  
  
Id returned from the HCAddWebSocketRoutedHandler call.  
  
  
### Return value  
Type: void
