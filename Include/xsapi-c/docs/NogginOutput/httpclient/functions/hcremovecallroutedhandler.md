# HCRemoveCallRoutedHandler  

Removes a previously added HCCallRoutedHandler.  

## Syntax  
  
```cpp
void HCRemoveCallRoutedHandler(  
         int32_t handlerId  
)  
```  
  
### Parameters  
  
*handlerId* &nbsp;&nbsp;\_In\_  
Type: int32_t  
  
Id returned from the HCAddCallRoutedHandler call.  
  
  
### Return value  
Type: void
